## ❓V-GIS가 뭐에요?
V-GIS는 Qgis에서 국토교통부 공간정보 오픈플랫폼인 브이월드(V-World)의 <br> 다양한 지도 서비스와 데이터를 손쉽게 활용할 수 있도록 돕는 도구입니다.

## ✨ 주요 기능
- **배경지도** — 브이월드 일반·항공·하이브리드 지도를 한 번에 추가
- **주제도(WFS)** — 국가 공간정보 주제도 검색·즐겨찾기·지도 추가, 원본 다운로드 바로가기
- **주소 검색** — 도로명·지번 주소 검색 후 지도로 이동, 최근 검색 기록 자동 저장
- **주소 조회(역지오코딩)** — 지도에서 클릭한 위치의 주소 확인
- **지오코딩** — 엑셀(CSV·XLS·XLSX) 주소 목록을 좌표로 일괄 변환 *(본인 API 키 필요)*
- **인코딩 변경** — 레이어 인코딩을 UTF-8·EUC-KR·CP949·MS949로 손쉽게 전환
- **폴리곤 스타일 변경** — 지적도 등 폴리곤 레이어 스타일·라벨 일괄 적용
- **포인트 일괄 매핑** — 좌표 목록을 입력해 포인트 레이어 생성
- **설정** — API 인증키 관리, 네트워크(HTTP/HTTPS) 방식, 툴바 표시 모드, 알림 방식 선택

## ⚙️ 요구사항
QGIS: 버전 3.10 이상

## 💾 설치 방법
저장소를 추가하여 이용하거나 <br> 소스코드를 다운로드 받아 직접 설치하는 방법을 선택하여 설치합니다.

### 방법 1: QGIS 플러그인 저장소 연동 (권장: 자동 업데이트)
연동 URL https://v-world.github.io/V-QGIS/plugin.xml

### 방법 2: ZIP 파일로 직접 설치
[💾다운로드 바로가기](https://github.com/V-world/V-QGIS/releases/)
